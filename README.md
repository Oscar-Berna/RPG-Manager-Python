# RPG Manager - Python

## Descripcion
Aplicacion de consola con arquitectura MVC conectada a MySQL.

## Requisitos
- Python 3.x
- mysql-connector-python
- XAMPP con MySQL corriendo

## Instalacion
pip install mysql-connector-python

## Base de datos
- Abrir XAMPP y iniciar MySQL
- Abrir phpMyAdmin en http://localhost/phpmyadmin
- Crear base de datos: rpg_manager
- Ejecutar el archivo db/schema.sql

## Ejecucion
python main.py
